# Контроль целостности файлов

Приложение с графическим интерфейсом для контроля целостности файлов с использованием хеш-алгоритмов CRC-32, MD5 и SHA-256. Позволяет создавать эталонный манифест и проверять изменения в директории.

## Возможности

- Выбор любой директории через графический интерфейс.
- Создание манифеста (JSON) с хешами всех файлов.
- Проверка целостности: обнаружение новых, удалённых и изменённых файлов.
- Использование трёх алгоритмов: CRC-32, MD5, SHA-256.
- Манифесты сохраняются в отдельной папке `manifests/` с указанием даты и имени папки.
- Кроссплатформенность (Windows, Linux, macOS).

## Скачать готовое приложение

Перейдите в раздел [Releases](https://github.com/Gnoly4/file-integrity-checker/releases) и скачайте файл для вашей операционной системы:

- `FileIntegrityChecker_windows.exe` – для Windows
- `FileIntegrityChecker_linux` – для Linux
- `FileIntegrityChecker_macos` – для macOS

## Как использовать

1. Запустите приложение (см. инструкции для вашей ОС ниже).
2. Нажмите **«Обзор...»** и выберите папку, которую хотите контролировать.
3. Нажмите **«Создать манифест»** – программа просканирует все файлы и сохранит эталонные хеши.
4. Внесите изменения в файлы (добавьте, удалите или измените содержимое).
5. Нажмите **«Проверить целостность»** – приложение покажет список изменений.

## Запуск приложения

### Windows

Просто дважды кликните по файлу `FileIntegrityChecker_windows.exe`. Если появится предупреждение безопасности, нажмите **«Дополнительно»** → **«Выполнить в любом случае»**.

### Linux

1. Откройте терминал в папке с файлом.
2. Дайте права на выполнение:
   ```bash
   chmod +x FileIntegrityChecker_linux
3. Запустите:
    ./FileIntegrityChecker_linux
Также можно настроить запуск двойным кликом через свойства файла (вкладка «Права» → «Разрешить выполнение»).

### macOS
1. Откройте терминал в папке с файлом.
2. Дайте права на выполнение и снимите атрибут карантина:
    chmod +x FileIntegrityChecker_macos
    xattr -d com.apple.quarantine FileIntegrityChecker_macos
3. Запустите:
    ./FileIntegrityChecker_macos
Если macOS блокирует запуск, зайдите в Системные настройки → Защита и безопасность → Общие и внизу нажмите «Всё равно выполнить». После этого двойной клик тоже будет работать.
